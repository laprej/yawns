Environment Variables

In case of nonstandard installation, it may be necessary to set the OMNETPP_TKENV_DIR environment variable so that Tkenv can find its parts written in Tcl script.

The default path from where the icons are loaded can be changed with the OMNETPP_IMAGE_PATH variable, which is a semicolon-separated list of directories that defaults to omnetpp-dir/images;./images.

The path from where Tkenv plugins written in Tcl are loaded can be changed with the OMNETPP_PLUGIN_PATH variable, which is a semicolon-separated list of directories that defaults to ./plugins.
